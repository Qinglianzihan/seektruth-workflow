# 求是工作流 (SeekTruth Workflow)

> 将《毛泽东选集》哲学方法论转化为 AI 编程的结构化工作纪律。

## 核心原则

1. **调查研究先行** — 没有调查就没有发言权
2. **矛盾分析驱动** — 每项任务只聚焦一个主要矛盾
3. **实践-认识-再实践** — 所有产出必须经过测试验证

## 快速开始

```bash
# 在现有项目中初始化求是工作流
npx seektruth-workflow init

# 启动一次新任务
stw start
```

## 五阶段工作流

| 阶段 | 名称 | 核心要求 |
| :--- | :--- | :--- |
| 1 | 调查研究 | 阅读代码，分析背景，产出《敌情分析报告》 |
| 2 | 抓住主要矛盾 | 锁定唯一核心任务 |
| 3 | 集中优势兵力 | 封锁无关文件，专注核心修改 |
| 4 | 实践检验 | 强制测试，失败则返回阶段 1 |
| 5 | 总结与转化 | 沉淀知识，产出《总结报告》 |
