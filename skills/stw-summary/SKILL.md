# STW Phase 5: Summary

Goal: convert completed work into reusable memory.

## Required Actions

1. Fill `.stw/Summary-Template.md` with:
   - task outcome
   - changed files
   - tests run
   - lessons learned
   - follow-up risks
2. Run:

```powershell
rtk stw next
rtk stw report
```

3. Only then say the work is complete.

## Keep It Short

Summaries should be factual and reusable. No narrative padding.
